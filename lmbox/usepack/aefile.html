# Aep模板扩展包的使用

TIP

使用前请激活并切换扩展包，具体的方法: 激活或切换扩展包

这种是默认的展示，也是全部的工具按钮，比较简单。

# Ae模板工具架详细介绍

  • 播放预览
  • 暂停预览
  • 将选中的预览项目导入Ae合成中，以播放头时间点为入点对齐，轨道自动选择空白轨道，防止覆盖已有的素材。
  • 管理和运行 jsx, jsxbin的脚本文件。

TIP

  • 导入的ae模板如果不是中文的，在时间轴的左侧将 源名称 切换 图层名称 即可。 脚本管理器是个实验室工具，所以在满足条件的情况下才使用

下面为共有按钮

  • 查看收藏的素材。
  • 调整预览播放音量（如有）。
  • 切换扩展包，安装扩展包，查看用户状态-

# Ae模板的导入

注意

导入前提是需要有一个有效时间轴者合成，并且是选中的状态。

  • 双击列表中的频谱预览即可快速导入。
  • 选择要导入的预览，然后点击工具架上的
    导入按钮也导入。
