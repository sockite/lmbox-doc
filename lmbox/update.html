# 2.1更新 2023.1.30

# 🚀 Features

  • 主要：增加了实验室工具“lut管理”，以后可以在扩展里面实现管理，预览，创建.cube文件啦。
  • 设置面板重做，增加了·主题色,提示音,顶部高度,心灵鸡汤,lut百分比,悬停播放,等等~~，这些设置的作用我会在 b 站出教程给大家介绍。
  • 增加了教程模块，目前这个版本只能发一些免费的公开课，希望大家多多支持！至于知识付费的，下个版本见。

# #### 🐞 Bug Fixes

  • 修复了ae脚本管理器的搜索功能
  • 修复了 静音剪辑的路径问题，现在可以使用任意的路径
  • 修复了 基本图形批量激活没有反应的问题
  • 修复了日志初始化的问题

# 🚧 Others

  • 静音剪辑在 cup 为inter的 MAC 电脑上一直因为没有测试环境，所以无法兼容，也没有得到验证。如果你有这种电脑，希望能测试一下，我也不知道现在的这个版本可不可以。

# 2.0更新 2023.1.30

🎉 🎉 🇨🇳新的一年，祝大家身体健康，心想事成！🇨🇳 🎉 🎉

# 🚀 Features

  • 按照承诺，实验室增加了==自建素材管理模块==，目前可以管理自己的音频素材库啦，支持 mp3 和 wav 的音频，如果有其他特殊格式，请留言给我，同样有动态的波形预览，可以推送到主页管理。此功能仅开放购买过本店音频扩展包的的用户使用。
  • 扩展样式全部重写，用户中心，实验室，设置，全部为侧边栏+右侧页面的方式，
  • 增加了会员权限预览页面，可以清楚的知道购买扩展包后会赠送那些附属的工具。
  • 增加了部分功能的帮助引导模块，可以点击右上角的小灯泡（如有）查看软件的使用步骤操作
  • 增加了教程页面（测试计划),本身我是一名讲师，所以后面会把教程同步到扩展内部。，

# 🚧 Others

  • 以前按钮的模板管理改名为用户中心了
  • 更改 主页底部的预览尺寸设置，列表预览，收藏按钮，工具提示，弹窗提示等样式，
  • 扩展底部的联系更加完善，更改为二维码和点击跳转的方式。， 如果你觉得这个项目对你有帮助，希望你能贡献一个 star⭐

# 1.9更新 2023.1.30

# 🚀 Features

-增加了主页的预览类型，现在支持四种，分别为：网格，列表，顶部预览+网格，顶部预览+列表。

# 🚧 Others

  • 更改了扩展包的初始化和持久化方式，可以记录切换，重启等永久记录最后一次的预览扩展包
  • 修复了音频的预览延迟问题。

  • 将日志模块挪到了设置面板
